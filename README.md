## Node.js web application hosted in Azure
A simple Node.js web application deployed to Azure using Docker and Terraform.

#### Overview
This project demonstrates a basic use case of the following methodologies:

- Docker containerization
- Infrastructure as Code with Terraform
- Cloud deployment on Azure App Service
- Version control with Git

#### Used Azure Resources

- App Service
- App Service Plan
- Container Registry

#### Visit the live application: https://nodejs-app-1337.azurewebsites.net/

#### Prerequisites
- Node.js
- Docker
- Git
- Azure CLI
- Terraform
